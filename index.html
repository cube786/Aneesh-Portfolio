Early Detection of Alzheimers Disease (College Project)

      Technology Stack

    • Python: Utilized Python for developing machine learning models and handling data preprocessing tasks.
    • Pandas: Employed for efficient data manipulation and preprocessing.
    • Scikit-learn: Implemented machine learning algorithms like Support Vector Machines (SVM) and Logistic Regression for classification.
    • Matplotlib: Used for visualizing performance metrics and algorithm comparisons.

    Key Features

  • Machine Learning Algorithms: Implemented SVM and Logistic Regression to classify patients as "Demented" or "Non-Demented" based on clinical data.
  • Preprocessing: Handled missing values, label encoding, and data splitting to prepare the dataset for training and testing.
  • Performance Metrics: Evaluated models using accuracy, precision, sensitivity, and specificity. The comparison revealed SVM as the more efficient algorithm for this dataset.
  • Disease Prediction: Successfully predicted Alzheimers disease based on the dataset's features, providing an early warning system for dementia.
  • Visualization: Compared the performance of different algorithms through bar charts to clearly present results.
